# Tuya IoTOS Embedded Mcu Demo  Ble Salty Spoon

[English](./README.md) | [中文](./README_zh.md)

## 简介 

本Demo通过涂鸦智能云平台、涂鸦智能APP、IoTOS Embeded MCU SDK实现一款能产生咸味的勺子。

已实现功能包括：

+ 电量检测

+ 频率控制

  



## 快速上手 

### 编译与烧录
+ 下载Tuya IoTOS Embeded Code

+ 执行test.uvprojx文件

+ 点击软件中的编译，并完成下载
